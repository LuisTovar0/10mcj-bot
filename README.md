# 10 McJ bot

This bot automates my daily task of formatting a message and some audio properties for [10 Minutes with Jesus](https://10minuteswithjesus.org/) that I send to the Telegram channel (now I just have to forward it) and the Signal group (now I just have to copy-paste it).

You can check out the [wiki](https://github.com/LuisTovar0/10mcj-bot/wiki) for the planning.
